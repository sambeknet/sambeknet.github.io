# sambeknet.github.io - Portfolio (v3.0)

This is a massive structural overhaul of my personal website! I completely redesigned the layout using a modern glassmorphism aesthetic and added a lot of dynamic new sections, widgets, and interactive features.

## Features
* **Glassmorphism Design:** Styled with frosted-glass containers, backdrop filters, and custom background gradients.
* **New Sections:** Added dedicated "Hobbies" and "Places" sections powered by Elfsight widgets and photo showcases.
* **Interactive Project Cards:** Project cards feature custom background preview assets that animate and shift when you hover over them.
* **Embedded Socials:** Integrated live LinkedIn profile badges, a GitHub stats card, and an interactive Linktree preview card with hover effects.
* **Dynamic Navigation:** A floating, centered navigation bar that automatically hides when you scroll down, plus a quick scroll-to-top button.
