# sambeknet.github.io - Portfolio (v2.0)

This is the second major iteration of my personal portfolio. The goal of this update was to introduce modern UI features like Dark Mode and dynamic animations, while thoroughly cleaning up the project's folder structure.

## Features
* **Dark/Light Theme Toggle:** A JavaScript-powered theme switcher that updates the background, text, and social icons.
* **Lottie Animations:** Uses lightweight `.json` animations for the header and project section backgrounds to create a dynamic feel without slowing down the site.
* **Centralized Assets:** All media files have been organized into a dedicated `assets/` folder to align with standard development practices.
* **Responsive Grid:** Projects are displayed using CSS Grid, ensuring they look great on both desktop and mobile devices.
